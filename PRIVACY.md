# Política de Privacidade

**Convert Money** - Aplicativo de Conversão de Moedas

Última atualização: 24/04/2026

## 1. Introdução

Este aplicativo é gratuito e sem fins lucrativos. Esta Política de Privacidade descreve como coletamos, usamos e protegemos suas informações.

## 2. Dados Coletados

O aplicativo **não coleta** dados pessoais dos usuários.

### 2.1 Dados de Uso

Não coletamos dados de uso, histórico de navegação ou qualquer informação sobre o comportamento do usuário no aplicativo.

### 2.2 Dados de Localização

Não coletamos dados de localização.

### 2.3 Informações do Dispositivo

Não coletamos informações do dispositivo (modelo, versão do sistema operacional, etc.).

## 3. APIs de Terceiros

O aplicativo utiliza as seguintes APIs públicas para obter taxas de câmbio:

- **ExchangeRate-API** (api.exchangerate-api.com)
- **CoinGecko** (api.coingecko.com)

Essas APIs são usadas apenas para buscar taxas de câmbio em tempo real quando você realiza uma conversão. Nenhuma informação pessoal é enviada a essas APIs.

## 4. Armazenamento de Dados

O aplicativo não armazena dados em servidores externos.

Todas as informações inseridas (valor a ser convertido, moedas selecionadas) permanecem apenas no dispositivo do usuário e são excluídas ao fechar o aplicativo.

## 5. Compartilhamento de Dados

Não compartilhamos, vendemos ou transferimos quaisquer dados pessoais para terceiros.

## 6. Publicidade

Este aplicativo não contém anúncios.

## 7. Compras no Aplicativo

O aplicativo é completamente gratuito e não oferece compras no aplicativo (IAP).

## 8. Direitos do Usuário

Você tem os seguintes direitos:

- **Acesso:** Não existindo coleta de dados, não há o que acessar.
- **Exclusão:** Os dados são mantidos apenas na memória do dispositivo.
- **Portabilidade:** Não aplicável.
- **Retirada de Consentimento:** Não aplicável, pois não utilizamos consentimento.

## 9. Crianças

O aplicativo é adequado para todas as idades e não coleta dados de crianças.

## 10. Alterações desta Política

Caso esta Política de Privacidade seja alterada,通知aremos através de uma atualização na descrição do aplicativo na loja.

## 11. Contato

Para dúvidas sobre esta Política de Privacidade, entre em contato através do desenvolvedor no repositório do GitHub.
